Extends Ember.Component
Defined in: packages/bricksui-form/lib/bu-editor.js:7
Module: bricksui-form
Parent Module: bricksui

Summary

基于ember与百度富文本编辑器umeditor 主要增加了数据绑定的功能，config配置项与umeditor一致 继承自Ember.Component，用法上与Ember.Component保持一致 使用方式如:

{{bu-editor value=title valueType="contentTxt"}}
