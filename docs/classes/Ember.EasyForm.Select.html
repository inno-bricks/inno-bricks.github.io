Ember.EasyForm.Select

Defined in: packages/bricksui-form/lib/chosen-select.js:6
Module: bricksui-form
Parent Module: bricksui

Summary

重置Ember.EasyForm下拉组建行为，将下拉框组件行为转换未chosen组建，用户通常无需直接使用该组建， 应通过Ember-EasyForm的方式使用该组件 可选的参数有 data-placeholder，表示下拉组建的占位符 使用方式如

{{#form-for model wrapper="bootstrap"}}
{{input author
as='select'
data-placeholder="请选择作者"
collection="category"
optionLabelPath="content.value"
}}
{{input tags as='select'
data-placeholder="请选标签"
multiple=true
collection="category"
optionLabelPath="content.value"
}}
{{/form-for}}
