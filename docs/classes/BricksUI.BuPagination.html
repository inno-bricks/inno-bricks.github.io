BricksUI.BuPagination

Extends Ember.Component
Defined in: packages/bricksui-thirdpart/lib/bu-pagination.js:6
Module: bricksui-thirdpart
Parent Module: bricksui

Summary

分页组件 分页组件负责视图的呈现,按钮的点击全部委托给pagerController,所以必须给分页组件 配置pagerController这个属性

用法

  • numberOfPages 显示的页码个数
  • pagerController 当前分页组件的控制器,必须配置,并且该控制器必须集成BricksUI.DynamicPageable
    {{bu-pagination numberOfPages=5 pagerControllerBinding="controller"}}
