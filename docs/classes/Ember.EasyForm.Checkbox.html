Ember.EasyForm.Checkbox

Extends Ember.EasyForm.Input
Defined in: packages/bricksui-form/lib/checkbox.js:5
Module: bricksui-form
Parent Module: bricksui

Summary

拓展Ember.EasyForm.Input的CheckBox功能 提供复选框按钮组的功能 使用方式如:
